## 참조와 복사
다른 변수에 복사할 수 있음.
```
var 값 = '2000'
var 복사 = 값
```

원시값(문자, 숫자, 불린)은 대입으로 복사할 수 있다.  
이게 객체일 때는 달라진다.  
```
var 사람 = {
	이름: '나'
{

var 복제인간 = 사람;
복제인간.이름 = '무지';
```

객체가 참조관계인지 보려면 비교연산자로.   
같으면 true.  
객체를 대입하면 참조관계가 됨.
